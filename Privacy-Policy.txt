Privacy Policy – Onexpo

Onexpo is a business-to-business ordering application for fragrance, dhoop, and related products. The application is accessible only to authorized users.

1. User Access

Users are provided with login credentials (username and password) directly by the Onexpo administration team. There is no self-registration feature.

2. Information Collection

We do not collect, store, or request any personal information such as:

Name
Phone number
Email address
Location data

User accounts are pre-created internally and only used for login access.

3. App Usage

The application allows authorized users to:

View product catalog
Check outstanding details
Add products to cart
Place orders
View order history
4. Data Storage

All data displayed in the app (such as products, orders, and outstanding amounts) is managed internally by Onexpo for business operations.

5. Data Sharing

We do not sell, share, or transfer user data to any third parties.

6. Security

Login credentials are provided securely by the administration. Users are responsible for keeping their credentials confidential.

7. Children’s Policy

This application is intended for business users only and is not designed for children under 13.

8. Changes

We may update this Privacy Policy if required. Any changes will be reflected in the app.
